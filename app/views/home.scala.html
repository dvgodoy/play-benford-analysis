A Single Approach to Different Detection Problems...

Benford Analysis is a suite for solving detection problems.

If you need to detect...

  • possible frauds in accounting records
  • edges or faint features in images
  • weak peaks in signals
  • spammers or other suspicious behaviors in social networks

.. then Benford Analysis is for you!

It includes a set of methods conveniently assembled in a package for Apache Spark and it is integrated with a Play Application to provide RESTful APIs for these methods.

Why Benford's Law?

Benford's Law is traditionally known as a tool for accounting audits. But its wide range of applications also include image and signal processing and social network analysis. Learn more about Benford's Law, its applications and how Benford Analysis implements its methods here.
